## Getting Started

1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/your-username/to-do.git
   ```
3. **Install dependencies** and run the project locally (see `README.md` for setup instructions)
4. Create a new branch for your work:
   ```bash
   git checkout -b feature/your-feature-name
   ```

---

## How to Contribute

- Browse open [Issues](../../issues) to find something to work on
- Comment on an issue to let others know you're working on it
- For new ideas, open an issue first before writing code

---

## Reporting Bugs

Found a bug? Please open an issue and include:

- A clear title and description
- Steps to reproduce the bug
- Expected vs actual behaviour
- Screenshots if applicable
- Your browser and OS

---

## Suggesting Features

Have an idea to improve To-do? Open a feature request issue and describe:

- What problem it solves
- How you imagine it working
- Any mockups or examples (optional)

---

## Code Contributions

1. Make your changes on your branch
2. Write clean, readable code with comments where necessary
3. Test your changes thoroughly in the browser
4. Commit with a clear message:
   ```bash
   git commit -m "Fix: correct task deletion bug"
   git commit -m "Add: filter tasks by date feature"
   ```
5. Push to your fork:
   ```bash
   git push origin feature/your-feature-name
   ```
6. Open a **Pull Request** against the `main` branch with a clear description of your changes

---

## Documentation

Good docs make the project better for everyone. You can contribute by:

- Improving the `README.md`
- Adding inline code comments
- Writing guides or usage examples

---

## Design Contributions

UI/UX improvements are welcome! You can contribute by:

- Suggesting layout or accessibility improvements via an issue
- Submitting updated CSS or design assets with your Pull Request
- Sharing mockups or wireframes in a feature request

---

## Code of Conduct

Please be respectful and inclusive in all interactions. We are committed to making this project a welcoming space for everyone.
